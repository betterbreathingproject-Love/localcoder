What's inside

Everything runs on your machine. Your code stays on your Mac.

🧠

Local LLM Inference

Run Qwen 3.6 35B and other models via MLX. Streaming responses, no API keys, no latency.

👁️

Vision Support

Drop in screenshots or mockups. The vision model describes what it sees and the agent acts on it.

🛠️

Agentic Tool Use

Reads/writes files, runs shell commands, searches code, browses the web — all autonomously.

📐

Spec-Driven Dev

Go from idea to code: requirements → design → tasks → implementation, with you in control at each step.

🔀

Task Orchestration

Complex work is broken into a task graph. The orchestrator runs tasks in parallel where possible.

🔭

LSP Safe Edits

Before writing code, the agent checks blast radius and verifies diagnostics via a live language server.

🌐

Browser Automation

Playwright lets the agent navigate, click, type, and screenshot — great for testing and scraping.

💾

Memory & Context

Long sessions are compacted automatically. A memory system recalls relevant context from past sessions.

⚡

Dual-Model Setup

A fast 0.8B model handles quick tasks (routing, vision, todos) while the big model does the heavy lifting.

🍎

Swift & Xcode

First-class support for iOS/macOS projects. Build, test, and debug Swift apps with Xcode integration.

💬

Chat & Vibe Mode

Just talk to it. Ask questions, brainstorm, or give it a task — it figures out the right mode automatically.

📱

Telegram Remote

Kick off jobs and monitor progress from your phone via the built-in Telegram bot integration.

How it works

Describe what you want. The agents handle the rest.

1

Describe your feature

Write what you want in plain language, or create a spec with requirements and design docs.

2

Tasks are generated

The agent breaks your feature into a task graph — a DAG of steps with dependencies and parallel paths.

3

Agents get to work

The orchestrator dispatches tasks to specialised agents. They explore, write code, run commands, and verify — all locally.

4

You stay in control

Watch progress in real time. Pause, inject context, or redirect at any point. Resume when you're ready.

Specialised agents

Each task is routed to the right agent automatically based on what needs doing.

Getting started

Up and running in a few minutes.

1

Download the DMG

Grab the latest release from GitHub Releases. Open it and drag to Applications.

2

Install a model

Download an MLX model to ~/.lmstudio/models/. We recommend Qwen3.6-35B-A3B-MLX-8bit for the primary model and Qwen3.5-0.8B-MLX-8bit as the fast model.

3

Launch and go

The setup wizard walks you through Python, model selection, and server start. Then open a project and start coding.
