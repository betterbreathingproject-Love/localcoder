Local AI-assisted coding for Apple Silicon — no cloud, no API keys, no limits.
Open Source — Built on the shoulders of giants
This project would not exist without the following open source projects. Thank you to every contributor.
⚡
MLX
Apple's array framework for machine learning on Apple Silicon. Powers all local inference.
MIT · Apple
🧠
mlx-lm
LLM inference and fine-tuning with MLX. Handles text-only models and tool calling.
MIT · Apple
👁
mlx-vlm
Vision-language model inference with MLX. Enables image analysis and screenshot understanding.
MIT · Prince Canuma
🗄
taosmd
Framework-agnostic AI memory system. 97% accuracy on LongMemEval-S. KnowledgeGraph, VectorMemory, and Zero-Loss Archive.
MIT · jaylfc
🔌
FastAPI
Modern Python web framework. Powers the local inference server at port 8090.
MIT · Sebastián Ramírez
🖥
Electron
Cross-platform desktop apps with web technologies. The shell that wraps everything together.
MIT · OpenJS Foundation
🎭
Playwright
Browser automation for agent tool use. Enables web browsing, screenshots, and UI testing.
Apache 2.0 · Microsoft
🔍
ast-grep
Structural code search and rewriting. Powers the AST-based code search tool.
MIT · Herrington Darkholme
📐
sentence-transformers
Semantic text embeddings. Powers vector memory search with all-MiniLM-L6-v2.
Apache 2.0 · UKP Lab
🚀
local-llm-autotune
Local LLM inference optimizer. Inspired the dynamic KV sizing, smart cache clearing, and prefill batching in our MLX server.
MIT · tanavc1
🔬
fast-check
Property-based testing for JavaScript. Used for correctness verification of all core modules.
MIT · Nicolas Dubien
🤗
Transformers
State-of-the-art ML models. Provides tokenizers and model utilities used throughout the stack.
Apache 2.0 · Hugging Face
🏗
Qwen3
The primary model family. Qwen3-30B-A3B for main inference, Qwen3.5-0.8B for fast assist tasks.
Apache 2.0 · Alibaba Cloud
📡
chokidar
File system watcher. Powers live file change detection and project watching.
MIT · Paul Miller
🐍
Pydantic
Data validation for Python. Used throughout the FastAPI server for request/response models.
MIT · Samuel Colvin
📦
Jinja2
Template engine for Python. Renders Qwen chat templates for tool calling support.
BSD · Pallets
Built with ❤️ for local-first AI development on Apple Silicon.
